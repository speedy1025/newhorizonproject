A lightweight Visual Studio Code clone built with a modern web stack. It replicates core editor functionality—multi-file editing, syntax highlighting, extensions support, and a customizable UI—while remaining fast, extensible, and easy to deploy.

Highlights

Core editor features: multi-tab file editing, split panes, minimap, search/replace, undo/redo
Language & tooling: syntax highlighting, basic IntelliSense-like completions, linting integration
Extensibility: plugin API for themes and extensions
UX: customizable themes, keyboard shortcuts, responsive layout
Dev-friendly: modular architecture and clear code structure for contributors
Tech stack

Frontend: Gemini Code assist and Claude ai
